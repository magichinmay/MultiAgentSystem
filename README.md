# Multi Agent System 

This is a Decentralized multi agent system, having different agents running on the SPADE framework, simulated in Gazebo using ROS2.

# Design
For implementation details, a research article is in progress of being published.
You can watch the video of the simulation [here](https://www.linkedin.com/posts/tarun-gattu123_gazebo-ros2-nav2-activity-7294297906029674496-33sy/)
